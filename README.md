# Order Coach

## 项目简介

本项目用于自动预约应县金土地驾校的科目二课程，旨在帮助学员更加快速便捷地练习。

* 程序运行的原理为发送http请求，与网页点击的效果一致。
* 本项目基于Github的Action功能运作。
